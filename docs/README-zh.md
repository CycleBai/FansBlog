# FansBlog

[English](/README.md) | Chinese

简单的静态博客生成器。
支持 Markdown 语法。

> [!IMPORTANT]
> 活跃开发阶段，大部分功能未实现，极其不稳定。
> 建议等待 1.0.0 发布后使用！

## 安装

```bash
npm i
```

## 使用

启动开发服务器：

```bash
npm run dev
```

编译成品：

```bash
npm run build
```

预览：

```bash
npm run preview
```
